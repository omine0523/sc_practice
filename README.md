# sc_practice

## 概要
sc_practiceは、シンプルな書籍管理アプリです。
書籍を一覧表示、検索、登録を行えます。

## 学習した内容
### PG改善
- コメントで処理の目的がわかるような記述を心がける。
- valueクラスを作成して画面から受け取った値を変換し画面とService層の責務を分ける。

### AOPの設定
- ログ出力：Service配下の全メソッドを対象にメソッド開始〜終了までの処理時間を計測・Sl4fjでlogbackにてログ出力する。
- トランザクション管理：DBへの処理内容によって分離レベル設定し、例外発生時にロールバックが実施されログを出力される。

### コネクションプールの設定
- あらかじめ複数のDB接続を保持し、アプリがDB接続を行う際に保持している接続情報を再利用してDB接続時の負荷や処理時間を削減する。
  → application.propatiesで設定

### SQL関連
- general_logの中のSQLクエリログ確認する。
  トランザクションが開始されたときのBIGIN、ロールバックのログを確認する。
- Springコンソールの設定を追加しMyBatisが発行するSQLをログ出力する。
